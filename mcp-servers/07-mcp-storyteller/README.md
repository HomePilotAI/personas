# mcp-storyteller

Production-ready MCP server for the **Storyteller** persona with support for persona live play sessions.

It exposes the following tools:

* **describe_session_vibe** – Creates a short vibe prompt (teasing, playful, romantic, etc.) for a live session.
* **build_branching_video_project** – Generates a branching AI video project with scenes, user choices, and endings.
* **suggest_live_play_examples** – Provides proven examples that are good starting points for interactive projects.

## Behavior highlights

- Persona-centered progression flow optimized for interactive video.
- Supports both image rendering mode (fast stills) and full video mode.
- Includes production-readiness checks in project output.
